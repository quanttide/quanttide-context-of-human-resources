# 渠道合作：决策背景梳理

## 业务特征

B2B 技术服务，主要吃资源（人脉、关系网、行业通路），而非标准化的销售话术或渠道铺量。

## 渠道分润 20-30% 的逻辑

### 本质：买资源

渠道比内部销售分润高（20-30% vs 10%），因为渠道带来的是**外部资源**——人脉、客户关系、行业通路。这些资源是公司自身不具备或不值得自建的，所以溢价购买。

### 为什么内部销售只有 10%

内部销售不只是拿 10% 提成，他们还有以下构成：

| 部分 | 说明 |
|---|---|
| 基础薪资 | 固定收入，兜底保障 |
| 10% 提成 | 基础业绩激励 |
| 内部贡献分配 | 公司自己获客的项目，内部按角色贡献再分配（商务配合 + 技术咨询等） |

10% 只是"基础提成"这一层。当项目是公司自获客时，参与该项目的内部角色（销售做商务配合初步报价、技术做咨询）会按照贡献额外分润。内部销售的整体收入是 **基础薪资 + 10% + 贡献分配**，而非只有 10%。

### 渠道 vs 内部销售的本质差异

| 维度 | 渠道合作伙伴 | 内部销售 |
|---|---|---|
| 提供什么 | 资源（人脉/通路） | 商务配合 + 初步报价 + 协调 |
| 分润方式 | 20-30%，纯风险收益 | 基础薪资 + 10% + 内部贡献分配 |
| 固定成本 | 无 | 有（基础薪资） |
| 可替代性 | 外部市场定价 | 有平台依赖，可被渠道替代 |

## 直觉背后的考量

1. **B2B 卖的是资源** — 客户不是看广告来的，是通过关系网触达的。渠道的价值在于他们拥有你够不到的关系网
2. **溢价买外部资源，平价用内部人力** — 内部销售是公司资产，值得培养和留存；外部渠道是消耗品，按单结算
3. **内部销售的真正价值不在"拉单"，而在"跟单"** — 商务配合、报价协调、技术对接才是内部销售的核心工作，提成只是其中一块

## 待确认的问题

- 内部贡献分配的具体机制是什么？谁定比例？
- 渠道遇到技术咨询时，内部技术团队如何配合？是否从渠道的分润中扣除？
- 渠道与内部销售跟同一个客户时，冲突如何处理？
